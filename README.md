# nodebots-hackaton

Simple counter for events.

## Running the server  

```bash
npm i
npm start
```

## Running the frontend separately  

```bash
cd frontend
http-server
```

## Exposing to the world  

```bash
ssh -R 80:localhost:3000 nodecounter@ssh.localhost.run
```

## Public Url  

https://nodecounter.localhost.run
